# 页面设计说明（桌面端优先）

## 全局规范

### Layout
- 桌面端优先（≥1280px）：页面内容最大宽度 1200px，居中；左右留白自适应。
- 主要布局：顶部导航 + 主内容区（卡片/表格列表）+ 轻量页脚。
- 响应式：
  - 1280px+：房间列表以表格+状态标签展示。
  - 768–1279px：表格列收敛为“房间信息 + 状态 + 操作”。
  - <768px：按需折叠（非优先），房间条目改为竖向卡片。

### Meta Information（统一默认）
- title 模板：`{页面名} - 联机大厅`
- description：`实时房间列表、快速加入、掉线/超时自动兜底。`
- Open Graph：
  - og:title 同 title
  - og:description 同 description
  - og:type: website

### Global Styles（Design Tokens）
- 背景色：#0B1220（深色）
- 卡片底色：#111B2E；边框：rgba(255,255,255,0.08)
- 主色（Primary）：#3B82F6；Hover：#2563EB
- 成功（Success）：#22C55E；警告（Warning）：#F59E0B；危险（Danger）：#EF4444
- 字体：系统字体栈；标题 20/24/28；正文 14/16
- 按钮：
  - Primary：实心主色，hover 加深
  - Secondary：描边按钮
  - Disabled：opacity 0.5 + 禁用指针
- Link：主色下划线 hover
- 动效：列表刷新与状态变化用 150–200ms 的淡入/颜色过渡（避免花哨动画）

## 页面 1：大厅（/）

### Page Structure
- 顶部导航（固定高度 56px）
- 主内容：两列结构（左：房间列表；右：快捷信息/说明）
  - ≥1280px：`grid-template-columns: 1fr 320px`
  - <1280px：右侧栏下移到列表下方

### Sections & Components
1) 顶部导航 NavBar
- 左侧：产品名「联机大厅」
- 右侧：
  - 昵称展示（未填写则显示“设置昵称”）
  - 刷新按钮（手动刷新房间列表，通常依赖实时更新）

2) 昵称入口 NicknameCard（必要但轻量）
- 位置：主内容区顶部（在房间列表上方）
- 元素：
  - 输入框：昵称（placeholder：如“玩家123”）
  - 按钮：保存
- 交互：
  - 保存后写入 localStorage，并用于加入房间与席位展示
  - 未设置昵称时，点击“加入”先弹出轻提示/对话框要求设置

3) 房间列表 RoomList
- 结构：表格（推荐）或列表（小屏）
- 列（桌面端）：
  - 房间号/名称
  - 人数：`current/max`
  - 状态：等待中 / 进行中
  - 已开局：是/否（用 Badge 表达）
  - 操作：加入按钮
- 状态标签：
  - 等待中：Success Badge
  - 进行中：Warning Badge
  - 满员/不可加入：Disabled（按钮置灰）
- 列表空态：
  - 文案：暂无可加入房间
  - 操作：刷新

4) 筛选与提示 FilterBar
- 筛选：全部 / 可加入（仅两个）
- 右侧提示：实时同步指示（小圆点 + “实时更新中”）

5) 右侧栏 SidePanel（桌面端）
- 卡片 1：规则简述（仅说明房间状态含义、掉线/超时会 AI 接管）
- 卡片 2：异常说明（掉线宽限期、超时默认动作）

## 页面 2：房间/对局（/room/:roomId）

### Page Structure
- 顶部导航（返回大厅、房间信息）
- 主内容：上方“房间状态条” + 下方左右分栏
  - 左：席位与状态
  - 右：回合/下注面板

### Sections & Components
1) 顶部导航 RoomNav
- 左：返回按钮（回到大厅）
- 中：房间号 + 状态 Badge（等待中/进行中）+ 人数
- 右：当前昵称、连接状态指示（在线/重连中）

2) 房间状态条 RoomStatusBar
- 信息：
  - 当前回合席位（轮到谁）
  - 倒计时（大号数字）
  - 系统提示（如“你已被 AI 接管/等待重连恢复”）
- 交互：倒计时临界（≤3s）颜色从中性渐变为 Warning

3) 席位面板 SeatsPanel
- 展示方式：2x2 或 1xN 卡片网格（根据 max_players）
- 每个席位卡片包含：
  - 昵称
  - 状态：在线 / 掉线 / AI
  - AI 标记：ai_controlled=true 时显示“AI 接管”Badge
  - 最近心跳：last_seen_at（可选仅用于调试/灰字）

4) 下注/行动面板 ActionPanel
- 当轮到自己：
  - 输入框：下注金额（数值）
  - 按钮：确认下注（Primary）
  - 倒计时提示：剩余秒数
- 非自己回合：
  - 文案：等待其他玩家操作
  - 仅展示已提交状态（如“已下注/未下注”）
- 超时后：
  - Toast：已执行默认动作 / AI 已接管
  - 面板进入只读

5) 事件与提示 Toast/Notice
- 加入成功、加入失败、房间满员、断线重连、超时处理、AI 接管等
- 统一右上角 Toast，3s 自动消失；重要告警（如被接管）可常驻 Banner

6) 重连恢复体验（关键状态）
- 断线时：页面顶部显示 Banner：“连接中…（可继续等待）”
- 恢复后：Toast：“已恢复连接，状态已同步”
- 若席位已被 AI 接管：Banner + 引导（例如“当前为观战状态/等待下一局”——具体规则由产品侧配置）
